Interaction notation for UX design

How to create a PiX

  1. Open the App
  2. Name your PiX and add a description
  3. Type inside each PiX cell, for adding an icon, just type pix- plus the icon name you wish to insert (check the complete list of icons)
    • Within the autocomplete, select with your arrow keys ↑ or ↓ and hit return to insert it
    • If you ignore the autocomplete and keep typing, just press space to insert the icon
  4. Split the score with the button to declare a different section or screen. You can name the new section in the textarea that appears on top
  5. For adding a new step in your score, just press tab and a new column (or step) will be added at the end, or the click on the button for inserting a new step wherever you need it
  6. For removing a step, just click on the button

Cómo crear una PiX

  1. Abre la Aplicación
  2. Nombra tu PiX y agrega una descripción
  3. Tipea dentro de cada celda de la partitura, para insertar un ícono sólo tipea pix- con el nombre del ícono a continuación (revisa la lista completa de íconos)
    • Aparecerá un menú con diferentes alternativas de acuerdo a la letra del ícono, para seleccionar el que quieres usar usa las flechas del teclado ↑ o ↓ y presiona return para insertarlo
    • Si prefieres ignorar este menú, sólo continua editando y presiona espacio para finalizar la inserción
  4. Para dividir la partitura en diferentes partes usa el botón para crear una línea vertical. Puedes nombrar la nueva sección en el nuevo espacio que aparece sobre la partitura
  5. Para agregar un nuevo paso o columna a la partitura, sólo presiona tab en la última celda y aparecerá otro paso a continuación. O puedes presionar el botón para insertar un espacio donde sea que lo requieras
  6. Para eliminar un paso o columna sólo presiona el botón
